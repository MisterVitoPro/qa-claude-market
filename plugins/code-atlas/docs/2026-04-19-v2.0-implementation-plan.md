# Code Atlas v2.0 Graph Language Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add semantic graph representation (graph-schema.json) and queryable graph language with `/code-atlas:query` skill to help Claude understand code dependencies at semantic level.

**Architecture:** Extend `/code-atlas:map` and `/code-atlas:update` to generate graph-schema.json with annotated nodes (modules/files with roles, criticality, stability, test coverage) and edges (dependencies with type, strength, directionality, impact). New `/code-atlas:query` skill lets Claude ask JSON-based questions (dependencies_of, dependents_of, filter, transitive_dependents) against the graph.

**Tech Stack:** TypeScript/Node.js (agents), JSON (artifacts), Markdown (skill definitions), existing Code Atlas v1.2 infrastructure
